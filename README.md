# FreshBox Website

เว็บไซต์ตัวอย่างสำหรับธุรกิจ **FreshBox** ผู้ให้บริการกล่องอาหารสดและอาหารสุขภาพ
ใช้เพื่อส่งงานอาจารย์เท่านั้น

---

## ฟีเจอร์หลักของเว็บไซต์

- หน้าแรก แสดงข้อมูลภาพรวมของธุรกิจ FreshBox
- หน้า About แสดงประวัติ แนวคิด และผู้บริหาร
- หน้า Services แสดงบริการ พร้อมตารางเปรียบเทียบบริการ
- หน้า Contact พร้อมแบบฟอร์มติดต่อ
- เมนูนำทางเชื่อมโยงทุกหน้า
